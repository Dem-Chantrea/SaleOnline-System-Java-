
# 🛒 Online Sale Processing System (Java)

## 📌 Project Description

This project is a **console-based Online Sale Processing System** developed in **Java**.
It simulates a simple e-commerce system with two roles:

* 👨‍💼 Admin
* 👤 Customer

The system allows product management, customer registration, order creation, and order processing.

---

## 🎯 Features

### 🔐 Admin(Functions)

* Login with username & password
* Add new products
* View product list
* Update product details
* Delete products
* View all orders
* Process orders

### 👤 Customer(Functions)

* Register new account
* Login
* View available products
* Create order
* View personal orders
* Cancel order

---

## 🧠 Data Structures Used (DSA)

This project applies important **Data Structures and Algorithms concepts**:

* **ArrayList** → Store Products, Customers, Orders
* **Linear Search** → Find users, products, and orders
* **Iterator** → Safe deletion of products
* **Basic OOP Design** → Class-based structure

---

## 🏗️ System Structure

* `Product` → Store product info (id, name, price, stock)
* `Customer` → Store user credentials
* `Order` → Store order details and status
* `OnlineSaleSystem` → Main system logic

## 📊 Sample Workflow

1. Admin logs in
2. Admin adds products with stock
3. Customer registers and logs in
4. Customer creates order
5. Admin processes order
6. Customer can cancel order (if not completed)

---

## ⚠️ Limitations

* No database (data stored in memory only)
* No GUI (console-based system)
* No advanced security (plain text password)
